                 Angular Blocks for the Compact Shell Proxy


                                       Round 2 analytic note


                                                Abstract
         The compact shell proxy can be reduced to an angular rounding defect Eang and a smooth
     positive payment H. This note attacks the remaining defect. First, an exact Stieltjes formula
     is given with the strict common-jump convention made explicit. The negative mean of every
     complete angular cell is combined with the single-peaked shape of q = −A′ to obtain continuous
     block estimates. A paired radial block argument then controls the full shifted sample sum,
     including all common jumps, by

     where N is the number of active shifted radial layers. On the actual compact residual N ≥ 1.
     The purely analytic replacement is therefore reduced to one smooth scalar inequality, displayed
     in (28). That scalar inequality is not proved here.

Remark 5 (Status and next step). The complete-cell bounds, the common-jump formula, and the
global estimate (23) are proved analytically. The ﬁrst remaining step in this route is the smooth
scalar inequality (28). Proving it will require a uniform lower bound for the positive outer-action
integral in (29), or, equivalently, a uniform upper bound for the boundary-layer loss in (31), together
with the explicit endpoint terms. Until that inequality is established, this note does not replace
the compact interval certiﬁcate.
